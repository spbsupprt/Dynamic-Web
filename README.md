# Dynamic-Web

Что нужно сделать?

Варианты стенда:

- nginx + php-fpm (laravel/wordpress) + python (flask/django) + js(react/angular);

- nginx + java (tomcat/jetty/netty) + go + ruby;

- можно свои комбинации.

Реализации на выбор:

- на хостовой системе через конфиги в /etc;

- деплой через docker-compose.
